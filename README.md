# Magento 2 Devbar
This module adds a floating bar to the frontend with useful information and shortcuts for developers.

> [!WARNING]  
> DO NOT ADD THIS MODULE TO A PRODUCTION ENVIRONMENT!

## Functionalities
- Info about Magento installation
  - Magento version
  - Database name
  - Search Engine
  - CSP mode
- Backend link
- Login with random user
- Add random (single) product to cart
- Checkout link
